# エラー調査ガイド

## 現在の状況

- サーバーは正常に起動している（ポート3001）
- `.next/server/app`ディレクトリが存在しない（ビルドが不完全な可能性）

## エラーの確認方法

### 1. ブラウザのエラーメッセージを確認

ブラウザで `http://localhost:3001` にアクセスし、以下を確認してください：

1. **開発者ツール（F12）を開く**
2. **Consoleタブ**でエラーメッセージを確認
3. **Networkタブ**で失敗しているリクエストを確認
4. **エラーメッセージの全文をコピー**

### 2. サーバーのログを確認

ターミナルに表示されているエラーメッセージを確認してください。

### 3. よくあるエラーの原因

#### A. モジュールが見つからないエラー
- `Cannot find module` または `Module not found`
- **原因**: インポートパスが間違っている、パッケージがインストールされていない
- **解決**: `npm install` を実行

#### B. CSSが読み込まれないエラー
- **原因**: PostCSSの処理が失敗している
- **解決**: `.next`を削除して再起動

#### C. ランタイムエラー
- **原因**: コード内のエラー
- **解決**: エラーメッセージに表示されているファイルと行番号を確認

## 次のステップ

ブラウザに表示されているエラーメッセージの全文を共有してください。それに基づいて具体的な解決策を提案します。
