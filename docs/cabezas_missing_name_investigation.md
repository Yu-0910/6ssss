# 2025年巨人所属「トレイ・キャベッジ（Trey Cabbage）」名前欠損調査レポート

## 📋 問題の概要

2025年から巨人に所属する「トレイ・キャベッジ（Trey Cabbage）」という選手の名前がデータに完全に欠損している。

## 🔍 調査結果

### 1. データ確認結果

#### CSVデータ（`_data/master_csv_calculated/batting_2025_CL_from_master.csv`）
- **巨人所属の選手数**: 58件
- **「Cabezas」「キャベッジ」「Cabbage」を含む選手**: **0件**
- **名前が完全に欠損している選手**: **0件**（すべての選手に名前が存在）

#### 生データ（`_data/master_csv/`）
- 2025年のCLデータファイルが存在しない（`batting_2025_CL_from_master.csv`が見つからない）
- 2024年以前のデータには「大勢（T.Taisei（outa）」という選手が`player_id: 81085155`で存在

#### HTMLキャッシュ（`output/html_cache/players/`）
- `81085155.html`は「大勢」という選手のページ（別の選手）
- 「Cabbage」「キャベッジ」を含むHTMLファイル: **見つからず**

### 2. 選手情報（Web検索結果より）

- **正式名称**: トレイ・キャベッジ（Trey Cabbage）
- **所属**: 読売ジャイアンツ（2025年から）
- **ポジション**: 外野手
- **前所属**: パイレーツ
- **身長**: 188cm、体重: 92kg、右投左打

### 3. 考えられる原因

#### 原因1: スクレイピング対象に含まれていない
- **可能性**: ⭐⭐⭐⭐⭐（非常に高い）
- **理由**: 
  - HTMLキャッシュに該当選手のページが存在しない
  - 2025年シーズン開始前の新規入団選手のため、スクレイピング時に未登録だった可能性

#### 原因2: 選手IDが割り当てられていない
- **可能性**: ⭐⭐⭐⭐（高い）
- **理由**:
  - 新規入団選手のため、NPBの選手IDがまだ割り当てられていない可能性
  - または、割り当てられたIDがスクレイピング対象リストに含まれていない

#### 原因3: 名前抽出ロジックの問題
- **可能性**: ⭐⭐（低い）
- **理由**:
  - 他の選手の名前は正しく抽出されている
  - ただし、外国人選手の名前表記（「トレイ・キャベッジ」）が特殊な形式の場合、抽出に失敗している可能性

#### 原因4: データ処理時の欠損
- **可能性**: ⭐（非常に低い）
- **理由**:
  - 他の選手のデータは正常に処理されている
  - 名前が完全に欠損している選手は他に見つからない

## 📝 推奨される対応

### 即座に実施すべき対応

1. **NPB公式サイトで選手IDを確認**
   - `http://npb.jp/bis/players/` で「トレイ・キャベッジ」を検索
   - 選手IDを特定

2. **HTMLキャッシュの確認**
   - 該当選手のHTMLページが存在するか確認
   - 存在しない場合、スクレイピングを再実行

3. **生データの確認**
   - スクレイピング後の生データに該当選手が含まれているか確認
   - 含まれている場合、名前抽出ロジックを確認

### 長期的な対応

1. **新規入団選手の自動検出機能**
   - シーズン開始前に新規入団選手を自動的に検出
   - 選手IDリストを定期的に更新

2. **名前抽出ロジックの改善**
   - 外国人選手の名前表記に対応
   - エラーハンドリングの強化

3. **データ検証機能**
   - 各年度・各チームの選手数を検証
   - 名前が欠損している選手を自動検出
