### 表示ルール（状況ごとの共通）

| 条件 | 扱い |
|------|------|
| 打数0 | 打率は「—」または非表示。長打率・OPSも同様 |
| 打席0 | 出塁率は「—」。四死球のみの打席は出塁率に含める |
| 打席・打数ともに少ない（目安: 打数<10） | 打率は参考値として表示するが、サンプル不足の注記を推奨 |
| 規定打席未到達 | 打率・OPS等の率指標は「規定未達」の注記を推奨 |
| 球種で打数0 | 打率は「—」。Whiff%も打席がなければ「—」 |
| ゾーンで打席0 | OPS・打率・HRは「—」 |
| 代打で打席数が極端に少ない | 打率より出塁率を重視。得点圏は参考程度 |

### 状況別・優先度の考え方

| 状況の種類 | 優先度 | 理由 |
|------------|--------|------|
| 通算・リーグ戦・交流戦 | 最優先 | 基本中の基本。全選手に必須 |
| 対チーム・球場・月別・週間 | 高 | スプリット分析の軸。対戦準備・調子把握 |
| ホーム/ビジター・デー/ナイター | 高 | 試合環境の影響。よく聞かれる指標 |
| 対左右・打順別・代打 | 高 | 起用判断の根拠。左殺し・打順適性 |
| カウント別・ランナー別 | 中 | 状況別の打撃。得点圏・追い込まれた時の成績 |
| 球種別・コース別・打球方向 | 中 | 投球配球・守備布陣の参考。データがやや特殊 |
| 盗塁マトリクス・本塁打種別 | 中 | 盗塁・クラッチに特化した分析 |
| 打撃内容・各種記録・起用 | 低〜中 | 補足情報。深掘り用 |
| セイバー・選球眼 | 低〜中 | 上級者向け。算出に手間 |
